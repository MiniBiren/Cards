# Cards

A local multiplayer card game for iOS.

## Features
- Host and join games on local network
- 2-8 players
- Customizable starting cards and decks
- Emoji reactions
- Spectator mode

## Requirements
- iOS 17.0+
- Xcode 15+

## Build
1. Clone the repo
2. Open `Cards.xcodeproj` in Xcode
3. Add Google Mobile Ads SDK via SPM
4. Build and run
